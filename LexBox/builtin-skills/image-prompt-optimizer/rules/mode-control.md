# Mode Control

- 文生图：从零建立完整画面，但先锁定主体、场景和构图，再补风格与完成度，不要写成超长杂糅列表。
- 参考图引导：先保留参考图的主体身份、核心轮廓、构图重心、主色关系、镜头关系，再补场景、光线、材质和完成度；最终提示词里必须逐张写出“参考图1/参考图2/...”各自负责什么。
- 图生图：先说明保留什么，再说明改什么。默认只做受控增强，避免替换主体、改脸、改服装主特征、改场景叙事。
- 如果参考图数量大于 1，优先抽取共同稳定特征；若存在冲突，服从用户文字中最明确的约束，再服从主体参考图的核心信息。
- 如果参考图数量大于 1，不要只写“参考图约束生效”或“参考图保持一致”；要明确逐张分工，例如“参考图1锁定主体身份，参考图2补充环境和构图，参考图3补充材质或服装细节”。
- 如果用户只是提供风格参考，不要误写成首尾帧或主体替换语气；要保留“风格借鉴但主体不变”的表达。
- 没有参考图时，不要伪装成参考图模式的描述。
