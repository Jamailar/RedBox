# Text Safety

- 除非用户明确要求画面中出现文字、logo 或招牌，否则不要把提示词原文、系统说明、布局标签、字体说明、配色说明、水印、AI 标签直接渲染进图里。
- “封面感 / 海报感 / 小红书配图 / 商业视觉”优先理解为留出版式空间和标题区，而不是直接在图片里生成一大段可见文字。
- 如果用户真的要求文字，优先保持文字极短、位置明确、与主体不抢视觉中心，同时仍然不要画出提示词说明、占位标签或无意义乱码。
- 不要把 “left panel / right panel / top area / bottom caption / blue theme / bold font” 这类说明词直接当作画面内容输出。
