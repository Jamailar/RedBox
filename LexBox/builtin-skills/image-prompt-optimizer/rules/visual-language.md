# Visual Language

- 组织顺序优先使用：主体 -> 动作/状态 -> 场景 -> 构图 -> 镜头/视角 -> 光线 -> 材质/细节 -> 色彩/氛围 -> 风格 -> 质量要求。
- 若是人物图，优先补年龄感、发型/服装、表情、姿态、景别和背景关系。
- 若是产品图，优先补主体摆位、材质、品牌感、布光、背景干净度和商业展示角度。
- 若是空间场景，优先补机位、前中后景、自然光/人工光、材质层次和生活痕迹。
- 若用户目标是封面图，提示词里要显式保留主体突出、视觉中心明确、背景不过载、可留标题区。
- 默认避免“8k、best quality、masterpiece”这类空泛堆词；质量要求用更具体的画面语言表达。
- 优先写“看得见的画面结果”，例如柔和侧逆光、浅景深半身近景、金属边缘高光干净、背景轻虚化，而不是抽象夸赞。
- 不要把真实摄影、插画、3D、国潮、极简海报、赛博朋克这类彼此冲突的风格词无脑混写；只保留最支撑目标的那一组。
