API Developer Guide

知识库导入接口

供浏览器扩展及第三方应用调用的本地 API，用于快速导入知识素材。

服务地址

http://127.0.0.1:31937/api/knowledge
鉴权：Header 携带 Authorization: Bearer {TOKEN} (如已配置)

核心接口

GET /health
检查服务状态及导入限制。
curl http://127.0.0.1:31937/api/knowledge/health
POST /entries
导入单条记录（网页、笔记、视频信息）。
{
  "kind": "link-article",
  "source": { "sourceUrl": "https://..." },
  "content": { "title": "标题", "text": "正文内容" },
  "options": { "allowUpdate": true }
}
POST /document-sources
导入本地文件夹或文档源。
{
  "kind": "tracked-folder",
  "rootPath": "/absolute/path/to/folder",
  "name": "我的文档库"
}

支持类型 (Kinds)

xhs-note wechat-article youtube-video link-article tracked-folder obsidian-vault

去重原则

优先级
dedupeKey > externalId > sourceUrl
静默更新
allowUpdate=true 时，匹配项将被覆盖。

响应格式

{
  "success": true,
  "entryId": "..."
}
