## Overview

**RedBox (RedConvert)** is a desktop AI workspace focused on the Xiaohongshu creation pipeline: capture, knowledge management, drafting, automation, and media generation.

This open-source version uses **user-configured AI providers**. It does not include hosted login, billing, or managed platform credentials.

## Core Features

1. Built-in Xiaohongshu browser and one-click capture
2. Local knowledge base and retrieval
3. Multi-workspace isolation
4. Random wander mode for idea generation
5. Manuscript editor with AI assistance
6. Advisor panel and group-chat collaboration
7. RedClaw single-session automation console
8. Image generation connected to the media library
9. Cover generation with template image + base image + title group
10. Scheduled and long-running background execution

## Quick Start

1. Download the installer from [Releases](https://github.com/Jamailar/RedBox/releases).
2. Open `Settings -> AI`.
3. Configure your own Endpoint / API Key / Model.
4. Test the connection and save.
5. Start from `Browser -> Capture -> Knowledge Base -> Manuscripts / RedClaw`.
