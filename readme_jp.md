## 概要

**RedBox (RedConvert)** は、Xiaohongshu の制作フローに特化したデスクトップ AI ワークスペースです。収集、ナレッジベース、執筆、自動化、メディア管理を一つにつなぎます。

このオープンソース版は**ユーザー自身が設定する AI プロバイダー**を使います。ホスト型ログイン、課金、管理済み認証情報は含みません。

## 主な機能

1. Xiaohongshu 内蔵ブラウザとワンクリック収集
2. ローカルナレッジベースと検索
3. ワークスペース分離
4. ランダムウォークによるアイデア生成
5. AI 補助付き原稿エディタ
6. アドバイザーとグループチャット協業
7. RedClaw 単一セッション自動化コンソール
8. 画像生成とメディアライブラリ連携
9. テンプレ画像 + ベース画像 + タイトル群による表紙生成
10. 定期・長時間バックグラウンド実行

## クイックスタート

1. [Releases](https://github.com/Jamailar/RedBox/releases) からインストーラをダウンロードします。
2. `設定 -> AI` を開きます。
3. 自分の Endpoint / API Key / Model を設定します。
4. 接続をテストして保存します。
5. `ブラウザ -> 収集 -> ナレッジベース -> 原稿 / RedClaw` の流れで使い始めます。
