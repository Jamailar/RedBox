## 개요

**RedBox (RedConvert)** 는 샤오홍슈 제작 흐름에 맞춘 데스크톱 AI 워크스페이스입니다. 수집, 지식베이스, 집필, 자동화, 미디어 관리를 하나로 연결합니다.

이 오픈소스 버전은 **사용자가 직접 설정한 AI 공급자**를 사용합니다. 호스팅 로그인, 결제, 관리형 자격 증명은 포함되지 않습니다.

## 핵심 기능

1. 내장 샤오홍슈 브라우저와 원클릭 수집
2. 로컬 지식베이스와 검색
3. 멀티 워크스페이스 격리
4. 랜덤 워더 모드 기반 아이디어 생성
5. AI 보조 원고 편집기
6. 어드바이저 패널과 그룹 채팅 협업
7. RedClaw 단일 세션 자동화 콘솔
8. 이미지 생성과 미디어 라이브러리 연동
9. 템플릿 이미지 + 베이스 이미지 + 제목 묶음 기반 커버 생성
10. 예약 및 장기 백그라운드 실행

## 빠른 시작

1. [RedBox 다운로드 페이지](https://redbox.ziz.hk/download) 에서 설치 파일을 다운로드합니다.
2. `설정 -> AI` 를 엽니다.
3. 자신의 Endpoint / API Key / Model 을 입력합니다.
4. 연결을 테스트하고 저장합니다.
5. `브라우저 -> 수집 -> 지식베이스 -> 원고 / RedClaw` 순서로 사용을 시작합니다.
