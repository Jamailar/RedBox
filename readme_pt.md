## Visão geral

**RedBox (RedConvert)** é um workspace de IA para desktop focado no fluxo de criação do Xiaohongshu: captura, base de conhecimento, escrita, automação e gestão de mídia.

Esta versão open source usa **provedores de IA configurados pelo usuário**. Não inclui login hospedado, cobrança nem credenciais gerenciadas pela plataforma.

## Recursos principais

1. Navegador Xiaohongshu embutido com captura em um clique
2. Base de conhecimento local e busca
3. Isolamento por múltiplos workspaces
4. Modo de exploração aleatória para gerar ideias
5. Editor de manuscritos com assistência de IA
6. Painel de conselheiros e colaboração em chat em grupo
7. Console de automação RedClaw por sessão
8. Geração de imagens integrada à biblioteca de mídia
9. Geração de capas com imagem de template + imagem base + grupo de títulos
10. Execução agendada e de longa duração em segundo plano

## Início rápido

1. Baixe o instalador em [Releases](https://github.com/Jamailar/RedBox/releases).
2. Abra `Configurações -> AI`.
3. Configure seu próprio Endpoint / API Key / Model.
4. Teste a conexão e salve.
5. Comece em `Browser -> Capture -> Knowledge Base -> Manuscripts / RedClaw`.
