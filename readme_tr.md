## Genel bakış

**RedBox (RedConvert)**, Xiaohongshu üretim akışına odaklanan bir masaüstü AI çalışma alanıdır: toplama, bilgi tabanı, yazım, otomasyon ve medya yönetimini tek zincirde birleştirir.

Bu açık kaynak sürüm **kullanıcının yapılandırdığı AI sağlayıcılarını** kullanır. Barındırılan giriş, faturalandırma veya platform tarafından yönetilen kimlik bilgileri içermez.

## Temel özellikler

1. Yerleşik Xiaohongshu tarayıcısı ve tek tıkla toplama
2. Yerel bilgi tabanı ve arama
3. Çoklu çalışma alanı izolasyonu
4. Fikir üretimi için rastgele gezinme modu
5. AI destekli taslak editörü
6. Danışman paneli ve grup sohbeti iş birliği
7. RedClaw tek oturum otomasyon konsolu
8. Medya kütüphanesi ile bağlı görsel üretimi
9. Şablon görsel + taban görsel + başlık grubu ile kapak üretimi
10. Zamanlanmış ve uzun süreli arka plan çalıştırma

## Hızlı başlangıç

1. Yükleyiciyi [Releases](https://github.com/Jamailar/RedBox/releases) sayfasından indirin.
2. `Ayarlar -> AI` bölümünü açın.
3. Kendi Endpoint / API Key / Model bilgilerinizi girin.
4. Bağlantıyı test edip kaydedin.
5. `Browser -> Capture -> Knowledge Base -> Manuscripts / RedClaw` akışından başlayın.
