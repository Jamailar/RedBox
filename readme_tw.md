## 專案概覽

**RedBox (RedConvert)** 是一個專注於小紅書創作流程的桌面 AI 工作台，打通採集、知識庫、創作、自動化與媒體管理全鏈路。

此開源版本使用**使用者自行配置的 AI 來源**，不包含官方代管登入、計費或托管憑證。

## 核心功能

1. 內建小紅書瀏覽器與一鍵採集
2. 本地知識庫與檢索
3. 多空間隔離
4. 隨機漫步靈感生成
5. 稿件編輯器 + AI 輔助
6. 智囊團與群聊協作
7. RedClaw 單會話自動化執行台
8. 生圖與媒體庫聯動
9. 封面圖生成（模板圖 + 底圖 + 標題組）
10. 後台定時 / 長週期任務執行

## 快速開始

1. 在 [RedBox 下載頁](https://redbox.ziz.hk/download) 下載並安裝。
2. 打開 `設定 -> AI`。
3. 填寫自己的 Endpoint / API Key / Model。
4. 測試連線並保存。
5. 從 `紅書 -> 採集 -> 知識庫 -> 稿件 / RedClaw` 開始使用。
